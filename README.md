# 📊 AI Study Pattern Analyzer

An intelligent web app that analyzes student study habits and provides insights using Machine Learning.

## 🚀 Features
- Study pattern analysis
- ML-based predictions
- Smart recommendations
- Interactive UI

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JS
- Scikit-learn
- Pandas, NumPy

## ⚙️ Run Locally
git clone https://github.com/your-username/AI-Study-Pattern-Analyzer.git
cd AI-Study-Pattern-Analyzer
pip install -r requirements.txt
python app.py
